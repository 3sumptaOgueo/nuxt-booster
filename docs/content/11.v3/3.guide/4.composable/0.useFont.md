# {{title}}

## Options

| Property   | Type      | Description                       | Default Value          |
| ---------- | --------- | --------------------------------- | ---------------------- |
| `critical` | `Boolean` | Override critical from component. | inherit from component |

```js
const { $getFont } = useFonts({critical: true});
```

## Return

| Property     | Description |
| ------------ | ----------- |
| `isCritical` |             |
| `$getFont`   |             |
